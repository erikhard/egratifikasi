Menampilkan data dengan datatables

by Okta Diantama · January 15, 2015

Pada artikel kali ini saya akan berbagi informasi mengenai cara menampilkan data. Jika pada artikel sebelumnya kita telah belajar bagaimana membuat paginasi disini dan disini, pada kesempatan kali ini kita akan belajar menampilkan data dengan menggunakan datatable. Dengan menggunakan datatables, kita tidak perlu membuat paginasi dan pencarian secara manual.

Berikut ini adalah contoh cara menampilkan data dengan datatables :

Step 1 : Anda perlu mendownload datatables, bootstrap, dan jquery. Anda bisa mendownload file-file tersebut di akhir artikel ini.

Step 2 : Saya asumsikan anda telah memiliki database dan sebuah tabel provinsi di dalamnya (anda juga bisa mendownloadnya di akhir artikel ini). Sekarang tinggal buat file index.php untuk menampilkan data dengan datatables. Berikut ini adalah kode untuk index.php.

<!doctype html>
<html>
    <head>
        <title>Harviacode - Datatables</title>
        <link rel="stylesheet" href="css/bootstrap.min.css"/>
        <link rel="stylesheet" href="css/dataTables.bootstrap.css"/>
    </head>
    <body>
        <div class="container">
            <table id="provinsi" class="table table-bordered">
                <thead>
                    <tr>
                        <th width="10%">Nomor</th>
                        <th width="15%">Provinsi</th>
                        <th width="15%">Aksi</th>
                    </tr>
                </thead>
                <tbody>
                    <?php
 
                    //Data mentah yang ditampilkan ke tabel    
                    mysql_connect('localhost', 'root', 'password-anda');
                    mysql_select_db('harviacode');
                    $sql = mysql_query('SELECT * FROM provinsi');
                    $no = 1;
                    while ($r = mysql_fetch_array($sql)) {
                    $id = $r['id'];
                    ?>
 
                    <tr align='left'>
                        <td><?php echo  $no;?></td>
                        <td><?php echo  $r['provinsi']; ?></td>
                        <td>
                            <a href="edit.php?id=<?php echo  $r['id_provinsi']; ?>">Edit</a> |
                            <a href="delete.php?id=<?php echo  $r['id_provinsi']; ?>">Delete</a>
                        </td>
                    </tr>
                    <?php
                    $no++;
                    }
                    ?>
                </tbody>
            </table>  
        </div>
        
        <script src="js/jquery-1.11.0.js"></script>
        <script src="js/bootstrap.min.js"></script>
        <script src="datatables/jquery.dataTables.js"></script>
        <script src="datatables/dataTables.bootstrap.js"></script>
        <script type="text/javascript">
            $(function() {
                $("#provinsi").dataTable();
            });
        </script>
    </body>
</html>

Demo | Download

 

Untuk membuat tabel dengan datatable ini sama dengan membuat tabel pada umumnya. Kita hanya perlu menambahkan id pada elemen table, kemudian membuat datatables dengan fungsi javascript. Tentu kita juga harus menginclude kan javascript dan css datatables.

Dengan cara di atas sebenarnya kita melakukan pengambilan seluruh data sekaligus, kemudian datatables yang membuat paginasinya. Jika data anda banyak misalnya mencapai puluhan ribu, teknik di atas akan terasa berat. Untuk mengatasi hal tersebut, kita bisa menggunakan datatables secara serverside.

Cara membuat datatables secara serverside akan kita bahas di sini.
